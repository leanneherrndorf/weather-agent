# Weather Agent

A daily global report on weather, seismic, and natural disaster events, generated by an AI agent and published as a React site.

https://leanneherrndorf.github.io/weather-agent/

**Stack:** Python · Claude (Anthropic) · Open-Meteo · NWS · USGS · Tavily · React · TypeScript · Vite

---

## Local Setup

### 1. Clone and create a Python virtual environment

```bash
git clone https://github.com/your-username/weather-agent.git
cd weather-agent

python3.11 -m venv .venv
source .venv/bin/activate      # macOS / Linux
# .venv\Scripts\activate       # Windows
```

> Python 3.14 ships without bundled pip and will fail. Use 3.11 or 3.12.
> If you don't have 3.11: `brew install python@3.11`

### 2. Install Python dependencies

```bash
.venv/bin/pip install -r requirements.txt
```

### 3. Configure environment variables

```bash
cp .env.example .env
```

Open `.env` and fill in your API keys:

```
ANTHROPIC_API_KEY=your-anthropic-api-key-here
TAVILY_API_KEY=your-tavily-api-key-here
```

- Anthropic key: https://console.anthropic.com/
- Tavily key (free tier, 1,000 searches/month): https://tavily.com/

### 4. Run the agent in test mode

```bash
.venv/bin/python3.11 -m agent.main --test
```

This runs the full agent loop and writes JSON to `site/public/data/`.

### 5. Install Node dependencies and run the dev server

```bash
cd site
npm install
npm run dev
```

Open http://localhost:5173 to view the site. The dev server hot-reloads on file changes.

### 6. Preview the production build locally

```bash
cd site
npm run build
npm run preview
```

### 7. Run Python tests

```bash
.venv/bin/python3.11 -m pytest tests/ -v
```

## Deployment (GitHub Actions + GitHub Pages)

1. Push this repo to GitHub
2. Go to **Settings → Secrets and variables → Actions** and add:
   - `ANTHROPIC_API_KEY`
   - `TAVILY_API_KEY`
3. Enable GitHub Pages: **Settings → Pages → Source → GitHub Actions**
4. The workflow (`.github/workflows/daily.yml`) runs automatically at 07:00 UTC each day — it runs the agent, builds the React site, and deploys to Pages in one job.

To trigger a test run manually: **Actions → Daily Weather Report → Run workflow**.

## Run Modes

The agent enforces a run guard. It will not run without an explicit mode:

| Mode | Command |
|------|---------|
| Test (local) | `.venv/bin/python3.11 -m agent.main --test` |
| Scheduled | `SCHEDULED_RUN=true python -m agent.main` (set automatically by Actions) |
| Manual (no flag) | Exits immediately with a usage message |
